# Example Use of Event Sourcing

This document outlines the plan for implementing an example event sourcing use case in `cfb-npm-template`. This example demonstrates how to use the event sourcing framework with a simple "last input logger" feature.

## Overview

**Feature**: Log the last input string entered by a user, with the ability to view it later.

**Architecture**:
- User enters text in UI → API Gateway → Lambda (appends event) → EventStore
- Event Publisher Lambda → EventBridge → Projection Listener Lambda → DynamoDB (projection)
- UI reads from projection via API Gateway → Lambda (reads projection) → DynamoDB

## Components

### 1. Backend Lambda: Log Last Input (Event Writer)

**Location**: `cfb-npm-template/functions/src/last-input-handler.js`

**Purpose**: Receives input string from API Gateway and appends an event to EventStore

**Functionality**:
- Receives POST request with `{ input: "string" }`
- **Requires Cognito authentication** - verifies access token from Authorization header
- Returns 401 Unauthorized if token is missing or invalid
- Uses `appendEvent()` from `@rinkkasatiainen/cfb-event-sourcing`
- Stream ID: `last-input-logger` (fixed for this example)
- Event Type: `InputLogged`
- Event Data: `{ input: string, timestamp: ISO string, userId: string }` (includes authenticated user ID)
- Returns the appended event

**Dependencies**:
- `@rinkkasatiainen/cfb-event-sourcing` package
- `@rinkkasatiainen/cfb-authorization-functions` package (for token verification)

**Environment Variables**:
- `EVENT_STORE_CLUSTER_ARN`
- `EVENT_STORE_SECRET_ARN`
- `EVENT_STORE_DB_NAME`
- `USER_POOL_ID` (Cognito User Pool ID for token verification)
- `AWS_REGION`

### 2. Backend Lambda: Projection Listener (EventBridge Trigger)

**Location**: `cfb-npm-template/functions/src/last-input-projection-handler.js`

**Purpose**: Listens to EventBridge events and updates DynamoDB projection with the last input

**Functionality**:
- Triggered by EventBridge Rule (listens to `InputLogged` events)
- Receives event from EventBridge
- Updates DynamoDB projection table with:
  - Key: `streamId` = `"last-input-logger"`
  - Attributes: `lastInput`, `lastEventNumber`, `updatedAt`
- Implements idempotency: checks if `eventNumber` already processed
- Only stores the last input (not history)

**Dependencies**:
- `@aws-sdk/client-dynamodb`
- `@aws-sdk/lib-dynamodb`

**Environment Variables**:
- `PROJECTION_TABLE_NAME` (DynamoDB table for last input projection)
- `AWS_REGION`

### 3. Backend Lambda: Get Last Input (Projection Reader)

**Location**: `cfb-npm-template/functions/src/get-last-input-handler.js`

**Purpose**: Reads the last input from DynamoDB projection and returns it via API Gateway

**Functionality**:
- Receives GET request
- Reads from DynamoDB projection table
- Key: `streamId` = `"last-input-logger"`
- Returns `{ lastInput: string, updatedAt: timestamp }` or `{ lastInput: null }` if no input yet

**Dependencies**:
- `@aws-sdk/client-dynamodb`
- `@aws-sdk/lib-dynamodb`

**Environment Variables**:
- `PROJECTION_TABLE_NAME`
- `AWS_REGION`

### 4. DynamoDB Projection Table

**Purpose**: Store the last input projection

**Schema**:
```typescript
{
  streamId: string (partition key),  // "last-input-logger"
  lastInput: string,                 // The last input string
  lastEventNumber: number,            // Last processed event number (for idempotency)
  updatedAt: string                   // ISO timestamp
}
```

**Key Features**:
- Single partition key: `streamId`
- Only stores the last input (not history)
- `lastEventNumber` used for idempotency checks

### 5. UI Web Component: Last Input Logger

**Location**: `cfb-npm-template/ui-components/src/components/cfb-last-input-logger.js`

**Purpose**: Web component that displays last input and allows submitting new input

**Features**:
- **Display**: Shows the last input (fetched from projection via API Gateway)
- **Input Box**: Text input for entering new input
- **Submit Button**: Submits new input to API Gateway
- **Loading States**: Shows loading while fetching/submitting
- **Error Handling**: Displays errors if API calls fail

**UI Structure**:
```html

    <label>Last Input:</label>
    <div id="last-input-value">Loading...</div>
  </div>

  </div>

```

**Ports**:
- `cfb-submits-last-input.js` - Port for submitting input (uses `authorizedFetch` from `@rinkkasatiainen/cfb-authorization`)
- `cfb-retrieves-last-input.js` - Port for retrieving last input (uses `authorizedFetch` from `@rinkkasatiainen/cfb-authorization`)

**Dependencies**:
- `@rinkkasatiainen/cfb-authorization` package (for `authorizedFetch`)

### 6. API Gateway Endpoints

**Endpoints**:
1. `POST /api/last-input` - Submit new input (calls `last-input-handler`)
2. `GET /api/last-input` - Get last input (calls `get-last-input-handler`)

**CORS**: Enabled for both endpoints

**Authentication**:
- `POST /api/last-input` - **Requires Cognito authentication** (verified in Lambda)
- `GET /api/last-input` - Optional authentication (can be public or authenticated)

#### Step 0.3: Create Auth Utilities
**File**: `cfb-authorization/functions/src/auth-utilities.js`

```javascript
import { CognitoIdentityProviderClient, GetUserCommand } from '@aws-sdk/client-cognito-identity-provider';

/**
 * Verify Cognito access token and return authorization status
 * @param {string} accessToken - The access token to verify
 * @param {string} userPoolId - The Cognito User Pool ID
 * @returns {Promise<boolean>} - True if token is valid, false otherwise
 */
export async function verifyAccessToken(accessToken, userPoolId) {
  if (!accessToken || !userPoolId) {
    return false;
  }

  try {
    const client = new CognitoIdentityProviderClient({});
    const command = new GetUserCommand({
      AccessToken: accessToken,
    });

    await client.send(command);
    return true;
  } catch (error) {
    // Token is invalid or expired
    console.warn('Token verification failed:', error.message);
    return false;
  }
}

/**
 * Extract access token from Authorization header
 * @param {Object} event - API Gateway event
 * @returns {string|null} - The access token or null if not found
 */
export function extractAccessToken(event) {
  const authHeader = event.headers?.Authorization || event.headers?.authorization;
  if (!authHeader) {
    return null;
  }

  // Extract token from "Bearer <token>" format
  const match = authHeader.match(/^Bearer\s+(.+)$/i);
  return match ? match[1] : null;
}

/**
 * Require authentication - verifies token and throws if invalid
 * @param {Object} event - API Gateway event
 * @param {string} userPoolId - The Cognito User Pool ID
 * @returns {Promise<{authorized: boolean, statusCode?: number, body?: string}>}
 */
export async function requireAuth(event, userPoolId) {
  const accessToken = extractAccessToken(event);
  
  if (!accessToken) {
    return {
      authorized: false,
      statusCode: 401,
      body: JSON.stringify({ error: 'Unauthorized: Missing access token' }),
    };
  }

  const authorized = await verifyAccessToken(accessToken, userPoolId);
  
  if (!authorized) {
    return {
      authorized: false,
      statusCode: 401,
      body: JSON.stringify({ error: 'Unauthorized: Invalid or expired token' }),
    };
  }

  return { authorized: true };
}
```

```javascript
import { appendEvent, getStreamVersion, createEventStoreConfigFromEnv } from '@rinkkasatiainen/cfb_-event-sourcing-first-draft';
import { requireAuth, extractAccessToken } from '@rinkkasatiainen/cfb-authorization-functions';
import { CognitoIdentityProviderClient, GetUserCommand } from '@aws-sdk/client-cognito-identity-provider';

const STREAM_ID = 'last-input-logger';
const EVENT_TYPE = 'InputLogged';

/**
 * Get user ID from access token
 */
async function getUserId(accessToken) {
  try {
    const client = new CognitoIdentityProviderClient({});
    const command = new GetUserCommand({ AccessToken: accessToken });
    const response = await client.send(command);
    return response.Username || response.UserAttributes?.find(attr => attr.Name === 'sub')?.Value || 'unknown';
  } catch (error) {
    console.warn('Failed to get user ID:', error.message);
    return 'unknown';
  }
}

export const logLastInputHandler = async (event) => {
  const headers = {
    'Content-Type': 'application/json',
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Headers': 'Content-Type,X-Amz-Date,Authorization,X-Api-Key',
    'Access-Control-Allow-Methods': 'POST,OPTIONS',
  };

  if (event.httpMethod === 'OPTIONS') {
    return { statusCode: 200, headers, body: '' };
  }

  // Require authentication
  const userPoolId = process.env.USER_POOL_ID;
  const authResult = await requireAuth(event, userPoolId);
  
  if (!authResult.authorized) {
    return {
      statusCode: authResult.statusCode || 401,
      headers,
      body: authResult.body || JSON.stringify({ error: 'Unauthorized' }),
    };
  }

  try {
    const body = JSON.parse(event.body || '{}');
    const input = body.input;

    if (!input || typeof input !== 'string') {
      return {
        statusCode: 400,
        headers,
        body: JSON.stringify({ error: 'Invalid input: expected string' }),
      };
    }

    // Get user ID from token
    const accessToken = extractAccessToken(event);
    const userId = await getUserId(accessToken);

    const eventStoreConfig = createEventStoreConfigFromEnv();
    const currentVersion = await getStreamVersion(STREAM_ID, eventStoreConfig);
    const expectedVersion = currentVersion;

    const appendedEvent = await appendEvent(
      STREAM_ID,
      EVENT_TYPE,
      {
        input,
        userId,
        timestamp: new Date().toISOString(),
      },
      expectedVersion,
      eventStoreConfig
    );

    return {
      statusCode: 200,
      headers,
      body: JSON.stringify({
        success: true,
        event: appendedEvent,
      }),
    };
  } catch (error) {
    if (error.name === 'ConcurrencyError') {
      return {
        statusCode: 409,
        headers,
        body: JSON.stringify({ error: 'Version conflict' }),
      };
    }
    console.error('Error in log last input handler:', error);
    return {
      statusCode: 500,
      headers,
      body: JSON.stringify({ error: 'Internal server error', message: error.message }),
    };
  }
};
```

#### Step 1.3: Create Projection Listener Handler
**File**: `cfb-npm-template/functions/src/last-input-projection-handler.js`

```javascript
import { DynamoDBClient } from '@aws-sdk/client-dynamodb';
import { DynamoDBDocumentClient, PutCommand, GetCommand } from '@aws-sdk/lib-dynamodb';

const STREAM_ID = 'last-input-logger';

export const lastInputProjectionHandler = async (eventBridgeEvent) => {
  const dynamoClient = DynamoDBDocumentClient.from(new DynamoDBClient({}));
  const tableName = process.env.PROJECTION_TABLE_NAME;

  try {
    const event = eventBridgeEvent.detail; // Event from EventBridge
    const { streamId, eventNumber, eventType, eventData } = event;

    // Only process InputLogged events for our stream
    if (streamId !== STREAM_ID || eventType !== 'InputLogged') {
      return { statusCode: 200, body: 'Event ignored' };
    }

    // Check idempotency: Get current projection
    const currentProjection = await dynamoClient.send(new GetCommand({
      TableName: tableName,
      Key: { streamId: STREAM_ID },
    }));

    // Skip if already processed
    if (currentProjection.Item && currentProjection.Item.lastEventNumber >= eventNumber) {
      console.log(`Event ${eventNumber} already processed`);
      return { statusCode: 200, body: 'Already processed' };
    }

    // Update projection with last input
    await dynamoClient.send(new PutCommand({
      TableName: tableName,
      Item: {
        streamId: STREAM_ID,
        lastInput: eventData.input,
        lastEventNumber: eventNumber,
        updatedAt: eventData.timestamp || new Date().toISOString(),
      },
    }));

    return { statusCode: 200, body: 'Projection updated' };
  } catch (error) {
    console.error('Error in projection handler:', error);
    throw error; // EventBridge will retry
  }
};
```

```javascript
import { DynamoDBClient } from '@aws-sdk/client-dynamodb';
import { DynamoDBDocumentClient, GetCommand } from '@aws-sdk/lib-dynamodb';

const STREAM_ID = 'last-input-logger';

export const getLastInputHandler = async (event) => {
  const headers = {
    'Content-Type': 'application/json',
    'Access-Control-Allow-Origin': '*',
    'Access-Control-Allow-Headers': 'Content-Type,X-Amz-Date,Authorization,X-Api-Key',
    'Access-Control-Allow-Methods': 'GET,OPTIONS',
  };

  if (event.httpMethod === 'OPTIONS') {
    return { statusCode: 200, headers, body: '' };
  }

  try {
    const dynamoClient = DynamoDBDocumentClient.from(new DynamoDBClient({}));
    const tableName = process.env.PROJECTION_TABLE_NAME;

    const result = await dynamoClient.send(new GetCommand({
      TableName: tableName,
      Key: { streamId: STREAM_ID },
    }));

    if (!result.Item) {
      return {
        statusCode: 200,
        headers,
        body: JSON.stringify({ lastInput: null, updatedAt: null }),
      };
    }

    return {
      statusCode: 200,
      headers,
      body: JSON.stringify({
        lastInput: result.Item.lastInput || null,
        updatedAt: result.Item.updatedAt || null,
      }),
    };
  } catch (error) {
    console.error('Error in get last input handler:', error);
    return {
      statusCode: 500,
      headers,
      body: JSON.stringify({ error: 'Internal server error', message: error.message }),
    };
  }
};
```

```javascript
import { authorizedFetch } from '@rinkkasatiainen/cfb-authorization';

const cfbRoot = '/api';

class CfbSubmitsLastInput {
  async submitInput(input) {
    if (!input || typeof input !== 'string') {
      throw new Error('Input must be a non-empty string');
    }

    try {
      const response = await authorizedFetch(`${cfbRoot}/last-input`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({ input }),
      });

      if (!response.ok) {
        const error = await response.json();
        throw new Error(error.error || `HTTP error! status: ${response.status}`);
      }

      return await response.json();
    } catch (error) {
      throw new Error(`Failed to submit input: ${error.message}`);
    }
  }
}

export default new CfbSubmitsLastInput();
```

```javascript
    this.#loadLastInput();
  }

  #render() {
    this.innerHTML = `
      <div class="last-input-display">
        <label>Last Input:</label>
        <div id="last-input-value" class="last-input-value">Loading...</div>
      </div>
      <div class="input-form">
        <input type="text" id="input-field" placeholder="Enter text..." />
        <button id="submit-button">Submit</button>
      </div>
      <div id="status-message" class="status-message"></div>
    `;

    this.#inputField = this.querySelector('#input-field');
    this.#submitButton = this.querySelector('#submit-button');
    this.#lastInputDisplay = this.querySelector('#last-input-value');
    this.#statusMessage = this.querySelector('#status-message');
  }

  #attachEventListeners() {
    this.#submitButton.addEventListener('click', () => this.#handleSubmit());
    this.#inputField.addEventListener('keypress', (e) => {
      if (e.key === 'Enter') {
        this.#handleSubmit();
      }
    });
  }

  async #loadLastInput() {
    try {
      this.#lastInputDisplay.textContent = 'Loading...';
      const lastInput = await cfbRetrievesLastInput.getLastInput();
      this.#lastInputDisplay.textContent = lastInput || '(no input yet)';
    } catch (error) {
      this.#lastInputDisplay.textContent = 'Error loading last input';
      console.error('Error loading last input:', error);
    }
  }

  async #handleSubmit() {
    const input = this.#inputField.value.trim();
    
    if (!input) {
      this.#showStatus('Please enter some text', 'error');
      return;
    }

    this.#submitButton.disabled = true;
    this.#showStatus('Submitting...', 'info');

    try {
      await cfbSubmitsLastInput.submitInput(input);
      this.#showStatus('Input submitted successfully!', 'success');
      this.#inputField.value = '';
      
      // Reload last input after a short delay (to allow projection to update)
      setTimeout(() => {
        this.#loadLastInput();
      }, 1000);
    } catch (error) {
      this.#showStatus(`Error: ${error.message}`, 'error');
    } finally {
      this.#submitButton.disabled = false;
    }
  }

  #showStatus(message, type) {
    this.#statusMessage.textContent = message;
    this.#statusMessage.className = `status-message ${type}`;
    
    if (type === 'success') {
      setTimeout(() => {
        this.#statusMessage.textContent = '';
        this.#statusMessage.className = 'status-message';
      }, 3000);
    }
  }
}
```

```typescript
    }));

    eventSourcing.grantEventStoreAccess(this.logLastInputLambda);

    this.lastInputProjectionLambda = new lambda.Function(this, 'LastInputProjectionLambda', {
      runtime: lambda.Runtime.NODEJS_22_X,
      handler: 'index.lastInputProjectionHandler',
      code: lambda.Code.fromAsset(
        path.join(__dirname, '../../cfb-npm-template/functions'),
        {
          bundling: {
            image: lambda.Runtime.NODEJS_22_X.bundlingImage,
            command: [
              'bash', '-c',
              'npm install && npm run build && cp -r dist/* /asset-output/',

          },

      environment: {
        PROJECTION_TABLE_NAME: this.lastInputProjectionTable.tableName,
        AWS_REGION: cdk.Stack.of(this).region,
      },
      timeout: Duration.seconds(30),
    });

    // Grant DynamoDB access
    this.lastInputProjectionTable.grantReadWriteData(this.lastInputProjectionLambda);

    });

    // Grant DynamoDB read access
    this.lastInputProjectionTable.grantReadData(this.getLastInputLambda);

    // EventBridge Rule: Trigger Projection Listener
    new events.Rule(this, 'LastInputProjectionRule', {
      eventBus: eventSourcing.eventBus,
      eventPattern: {
        source: ['event-store'],
        detailType: ['EventStored'],

          eventType: ['InputLogged'],

      description: 'API for npm-template event sourcing example',

        allowOrigins: apigateway.Cors.ALL_ORIGINS,
        allowMethods: apigateway.Cors.ALL_METHODS,
        allowHeaders: ['Content-Type', 'X-Amz-Date', 'Authorization', 'X-Api-Key'],
      },
    });

    // POST /api/last-input
    const lastInputResource = this.api.root.addResource('last-input');
    lastInputResource.addMethod(
      'POST',
      new apigateway.LambdaIntegration(this.logLastInputLambda)
    );

    // GET /api/last-input
    lastInputResource.addMethod(
      'GET',
      new apigateway.LambdaIntegration(this.getLastInputLambda)
    );
  }
}
```
